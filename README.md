## Description
"Wherever Forever" is a project to digitize the personal travel diaries of Serena, a travel blogger, and create an engaging travel blog that highlights places, activities, and unforgettable experiences. The blog allows Serena to share her stories and experiences with a global audience, featuring personal recommendations and reflections.

## Features
* **Interactive Blog Feed**: Developed an interactive blog feed page with a carousel for the latest posts.
* **Detailed Blog Posts**: Implemented a public blog post page displaying detailed post information.
* **Editable Posts**: Created an editable blog post page for the owner, with functionality to update or delete posts.
* **User Authentication**: Built a login page for user authentication.
* **User Registration**: Added a registration page for future team members.
* **Responsive Design**: Ensures the blog is accessible and looks good on all devices, including desktops, tablets, and mobile phones.
* **API Integration**: Uses API requests for creating, updating, and deleting blog posts.
* **Secure Token Storage**: Saves authentication tokens in the browser for secure post management.
